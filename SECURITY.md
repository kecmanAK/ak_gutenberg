# Security Policy

## Supported versions

Security fixes are applied to the latest public version maintained in this repository.

## Reporting a vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report issues privately to the repository owner and include:

- a short description of the issue
- reproduction steps
- affected WordPress and PHP versions
- proof-of-concept details if relevant

The goal is to validate the report quickly and coordinate a fix before public disclosure.
