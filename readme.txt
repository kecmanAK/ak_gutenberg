UX-focused Gutenberg blocks that help non-technical WordPress editors publish rich layouts without developer support.

== Description ==

AK Gutenberg Custom Blocks is a production-oriented custom block collection for WordPress projects that need richer layouts and a clearer editorial workflow inside Gutenberg.

Its strongest differentiator is editor UX. The plugin was designed so content editors who are not especially comfortable with CMS tooling can still publish independently and with confidence, without needing repeated developer help for routine content updates.

Included blocks:

* Logo Grid
* Photo With Text
* 2 Photos + Title (Left / Right)
* 2 Photo Blocks + Pop-Up Title
* Masonry Photo Layout - 2 Col
* Image Layout 1
* Image Layout 2
* 2 Horizontal Videos
* 2 Videos + Title (Left / Right)
* Masonry Video Layout - 2 Col
* Masonry Video Layout - 3 Columns

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or upload the plugin ZIP through the WordPress admin.
2. Activate the plugin through the `Plugins` screen in WordPress.
3. Open the block editor and insert the custom blocks from the block inserter.

== Frequently Asked Questions ==

= Who is this plugin for? =

Teams that want a more guided Gutenberg editing experience, especially when content editors should manage page updates without recurring developer involvement.

= Is this intended only for local development? =

No. This repository is prepared for GitHub-based distribution and production use as a final plugin package.

= What is the main product advantage? =

The plugin emphasizes editor usability so non-technical users can work more independently in WordPress.
